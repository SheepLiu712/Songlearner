# 音乐清洗工作流

1. 利用分离伴奏模型得到人声。示例代码`python inference.py --model_type bs_roformer --config_path configs\viperx\model_bs_roformer_ep_317_sdr_12.9755.yaml --start_check_point pretrain\model_bs_roformer_ep_317_sdr_12.9755.ckpt --input_folder inputs\wavs --store_dir outputs`
2. 对分离得到的人生进行去噪，示例代码`python inference.py --model_type mel_band_roformer --config_path configs\model_mel_band_roformer_denoise.yaml -
-start_check_point pretrain\dereverb_mel_band_roformer_anvuew_sdr_19.1729.ckpt --input_folder inputs\vocals --store_dir outputs`
3. 对得到的wav编码为mp3，使大小降低到8MB以下。
4. 使用MSAF对得到的伴奏文件的mp3进行分段，得到每段的起止时间。
5. 清洗lrc文件，使用分隔符---将歌词分段。
6. 使用LLM进行精细分段；
7. 将得到的分段信息和清洗后的歌词进行对齐，生成最终的分段文件。
8. 检查分段文件的质量，必要时进行人工调整。

注意：
1. 本项目按照requirements.txt中的依赖进行开发，但是MSAF是一个很久的项目，依赖旧的scipy接口。报错时进入MSAF的代码中，找到scipy相关的部分，按照报错提示修改接口调用即可。
2. 本项目需要一个QWEN的API KEY，放在环境变量中，命名为QWEN_API_KEY。
